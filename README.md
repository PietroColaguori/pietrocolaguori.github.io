### Portfolio
- *Smart Contract fuzzing using Echidna* \
Given an insecure smart contract written in [Solidity](https://soliditylang.org/) I had to use the fuzzing tool [Echidna](https://github.com/crytic/echidna) to discover the vulnerabilities and correct them according to a given set of requirements. \
You can find the code and final report [here](https://github.com/PietroColaguori/Ethereum-Fuzzing).
- *Pseudo-Malloc* \
A simple replacement for the libc function `malloc` using a buddy allocator and `mmap`. \
You can find the code [here](https://github.com/PietroColaguori/Pseudo-Malloc).
- *Anomaly detection on grapes* \
I trained an [autoencoder](https://en.wikipedia.org/wiki/Autoencoder) to distinguish between images of diseased berries and healthy ones. This project is also my B.E. final thesis. The data was unlabeled and this is an example of [unsupervised learning](https://en.wikipedia.org/wiki/Unsupervised_learning). \
You can find the code [here](https://github.com/PietroColaguori/Anomaly-Detection-Grapes).
